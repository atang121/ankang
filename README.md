# 安康血压记录

一个简洁温馨的血压记录 Web 应用，匹配域名 **ankang.site**，可部署到任何静态服务器或直接打开使用。

> 「安康」寓意平安健康——祝您安康，从此刻的每一次记录开始。

## 功能特性

- **快捷记录** - 输入三个数字或粘贴语音识别内容，自动提取血压数值
- **手动填写** - 逐项输入高压、低压、脉搏等详细数据
- **智能解析** - 支持中文数字（如「一百二十五 八十」）和语音转文字内容
- **补录历史** - 可以补录过去日期的血压记录
- **趋势图表** - 查看近7天、30天或自定义区间的血压趋势，时间轴无数据天也显示
- **趋势结论** - 自动显示阶段结论、平均值、偏高天数、偏高记录、缺测天数和上一周期对比
- **每日热力** - 用颜色标出正常、临界、偏高和未测量日期
- **演示数据** - 管理员演示模式可生成/清除最近 30 天模拟血压数据
- **触摸交互** - 图表支持触摸/点击查看具体数据气泡提示
- **历史查询** - 快捷筛选近7天/30天/3个月，或自定义日期范围
- **数据备份** - 导出 JSON 格式备份，支持恢复，7天未备份自动提醒
- **本地提醒** - 明确提醒数据仅保存在当前浏览器，建议定期备份
- **安装引导** - 支持安卓、苹果、华为和内置浏览器的添加桌面说明
- **意见反馈** - 可配置飞书表单/多维表格表单链接
- **匿名统计** - 可配置访问、按钮点击和记录时间段统计，不上传血压值、备注或日期
- **PWA 支持** - 可添加到手机主屏幕，支持离线使用

## 部署方式

### 方法一：直接打开
直接用浏览器打开 `index.html` 文件即可使用。

### 方法二：本地服务器
```bash
# Python
python -m http.server 8080

# Node.js
npx serve .

# PHP
php -S localhost:8080
```

### 方法三：Cloudflare Pages + Worker

前端静态页面部署到 Cloudflare Pages，Worker 负责把匿名统计和意见反馈写入飞书多维表格。

Worker 配置在 `worker/wrangler.toml`，其中 Base Token 和 Table ID 可以公开提交；`LARK_APP_ID`、`LARK_APP_SECRET` 必须通过 Cloudflare Worker Secrets 设置，不能写进前端代码或仓库。

## 数据存储

所有数据存储在浏览器的 `localStorage` 中：
- `bp_records_v1` - 血压记录数据
- `bp_last_backup_v1` - 上次备份时间
- `bp_anon_session_v1` - 匿名统计会话 ID，不包含健康数据
- `bp_local_notice_seen_v1` - 本地存储提醒是否已展示
- `bp_standard_v1` - 用户自定义的单次偏高提醒值

**注意**：清除浏览器数据会丢失记录，请定期使用「备份数据」功能导出备份。

## 判定标准

默认单次偏高提醒为 `140/90`，设置页可点击「自定义」按医生建议调整。家庭均值关注值保留为 `135/85` 展示，不提供普通自定义入口，避免把周期均值提醒误当成诊断标准。

## 匿名统计

默认不会发送任何统计或反馈请求。如需接入 Cloudflare Worker，在部署页面前设置：

```html
<script>
window.ANKANG_API_BASE = 'https://api.ankang.site'
</script>
```

前端会向以下接口发送请求：

- `POST /collect`：匿名统计事件
- `POST /feedback`：用户主动填写的意见反馈

统计 payload 只包含事件名、匿名 session、版本号和白名单字段（页面、按钮动作、筛选周期、结果），不会上传血压值、脉搏、备注、具体测量日期或自定义日期范围。

记录保存事件会额外上报粗粒度时间段 `time_bucket`：
- `morning`：05:00-09:59
- `noon`：10:00-12:59
- `afternoon`：13:00-17:59
- `evening`：18:00-21:59
- `night`：22:00-04:59

不会上传具体测量时间。

统计后台的查看权限不在这个静态 H5 里实现，需要在你选择的统计服务中设置管理员账号和访问权限。

## 意见反馈

意见反馈在网页内弹窗填写，提交到 `ANKANG_API_BASE + '/feedback'`，由 Worker 写入飞书多维表格 `反馈记录`。反馈不会自动附带血压值、脉搏、测量日期或备注。

不要把飞书应用密钥、邮箱 SMTP 密码等敏感信息放在前端代码中；这些参数应放在 Worker Secrets / 环境变量里。

## 管理员演示模式

静态 H5 不能提供真正保密的管理员权限。当前管理员演示模式只用于本机调试：访问 `index.html?admin=1` 后，设置页会显示「管理员演示工具」，可以生成最近 30 天模拟数据，也可以只清除模拟数据。

模拟数据会标记 `isMock: true`，清除时不会删除真实记录。

## 添加到主屏幕

### iOS (Safari)
1. 打开网站
2. 点击分享按钮
3. 选择「添加到主屏幕」

1. 打开网站
2. 点击菜单（三个点）
3. 选择「添加到主屏幕」或「安装应用」

## 技术栈

- 纯 HTML/CSS/JavaScript（无依赖）
- Canvas API（图表绘制 + 触摸交互）
- LocalStorage（数据持久化）
- Service Worker（离线支持）
- PWA（渐进式 Web 应用）
